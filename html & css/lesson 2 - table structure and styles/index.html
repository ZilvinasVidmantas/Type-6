CSS - stiliai

CSS - pakopiniai stiliaus šablonai (angl.: casscadian style sheets). Tai kompiuterinė kalba skirta stilizuoti HTML turinį.

CSS rašomas tokia tvaka: ieškome html elementų, naudodami selektorių, ir tarp garbanotų skliaustų aprašome stilius, tiems elementams kurie buvo surasti pagal selektorių.

Stiliaus savybės rašomos tokia tvarka:
  stiliaus-savybės-pavadinimas: stiliaus-savybės-reikšmė;

Stiliaus šablonų pavyzdžiai

                selektorius {
                    savybės-pavadinimas1: savybės-reikšmė1;
                    savybės-pavadinimas2: savybės-reikšmė2;
                    savybės-pavadinimas3: savybės-reikšmė3;
                }

                Pasirinkti visus h1 elementus ir nustatyti jam tarp garbanotų skliaustų aprašytas savybes
                h1 {
                    font-size: 30px;
                    color: red;
                    margin-top: 10px;
                    margin-bottom: 10px;
                }

Box model

Box-model tai modelis pagal kurį yra sudarytas kiekvienas HTML elementas. Elementai yra stačiakampiai, ir turi pagrindines 4 sekcijas (pradedant nuo vidurio į išorę):

  • content - vieta kurią užima elemento vaikai/turinys
  • padding - elemento praplėtimas, kuriam galioja nustatoma foninė spalva
  • border - elemento sienelės
  • margin - elemento paraštės

Box-model pratybos

  • margin
  • padding
  • display
  1. Sukurti kvadarčiuką, kurio išmatavimai būtų 100x100, o fonas raudonas. Kvadračiuko stilius aprašyti, naudoti klasę "square". Kvadračiukai turi būti "inline-bloka'ai"
  2. Sukurti klases, kurios prideda margin į visas puses, tik į dešinę, tik į kairę, tik į apačią ir tik į viršų po 20px. Jų pavadinimai turi būti tokie: m-20, mt-20, mr-20, ml-20, mb-20. Sukūrus šias klases, pritaikyti kvadračiukams, ir peržiūrėti su developer tools, kaip buvo pritaikytos paraštės
  3. Sukurti klases, kurios prideda padding į visas puses, tik į dešinę, tik į kairę, tik į apačią ir tik į viršų po 20px. Jų pavadinimai turi būti tokie: p-20, pt-20, pr-20, pl-20, pb-20. Sukūrus šias klases, pritaikyti kvadračiukams, ir peržiūrėti su developer tools, kaip buvo pritaikytas kamšalas. Sukurkite 5 kvadaračiukus, kuriems pritaikysite po vieną iš "padding" klasių. Sarašas

  • Sukurkite 'ul' sąrašta, su nuorodomis į jūsų 5 mėgstamiausius puslapius
  • Padarykite, jog kiekviena eilutė būtų skirtingos spalvos, tam naudokite klases tokius principu: text-{spalvos pavadinimas}, pvz.: .text-black, text-red ir t.t.
  • sukurkite klasę pavadinimu 'mb-10', kuri nustatytų 10px paraštę į apačią. Šią paraštę pritaikykite visiems 'li' elementams, išskyrus paskutiniui.
  • Suraskite intenete būdą, kaip padaryti, jog visi sąrašo elementai (li), išskyrus paskutinį, turėtų apačioje 10px paraštę. Tai turi veikti nepriklausomai nuo sąrašo elementų skaičiaus.

Nuotraukų koliažas

Sukurkite 5 nuotraukas, kad jos visuomet tilptų į vieną eilutę

  • Nuotraukų aukštis 200px (nepriklausomai nuo jų orginalaus dydžio)
  • Kiekvienos nuotraukos plotis 15% tėvinio elemento pločio
  • Klasę kurti tik tėviniui nuotraukų elementui: 'img-row'. Galite naudoti hierarchinius selektorius
  • Pabandykite surasti būdą, jog kiekviena nuotrauka būtų 20% tėvinio elemento pločio
